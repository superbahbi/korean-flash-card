# Goyo: Zen-Inspired Korean Learning App

Goyo (고요 - "Stillness") is a modern language learning application that combines **Learning Science** with **Zen Minimalism** to help you master Korean vocabulary effectively and consistently.

## ✨ Key Features

- **Spaced Repetition System (SRS)**: Optimized review cycles based on memory research.
- **Multisensory Learning**: Integrated Text-to-Speech (TTS) for native-like pronunciation.
- **Gamified Experience**: Streak tracking, XP, and Leveling to keep you motivated.
- **Active Recall**: Varied practice modes focusing on retrieval practice.
- **Zen Design**: A clean, distraction-free interface inspired by Korean aesthetics.

## 🧬 Learning Science Foundation

This app is built on three scientifically-validated methodologies:
1. **Spaced Repetition**: Expanding intervals to maximize long-term retention.
2. **Active Recall**: Forcing the brain to reconstruct knowledge rather than passive recognition.
3. **Dual Coding**: Combining visual (Hangul) and auditory (TTS) input for stronger neural pathways.

## 🎮 Gamification & Habits

- **Daily Streaks**: Build a habit by practicing every day.
- **XP & Levels**: Earn experience points for every card reviewed.
- **Daily Goals**: Bite-sized learning targets (10 cards/day).

## 🚀 Getting Started

1. Clone the repository.
2. Install dependencies: `pnpm install`.
3. Start the development server: `pnpm dev`.
4. Open your browser and start your Korean journey!
